# 🎲 Guess My Number

A fun number guessing game built with **HTML, CSS, and JavaScript**.  
The player tries to guess a secret number between **1 and 20** within limited attempts.

---

## ▶️ How to Play

1. Enter a number between **1–20**.
2. Click **Check!** to see if you guessed right.
3. You win if your guess is correct 🎉, otherwise you get hints (Too High / Too Low).
4. Each wrong guess reduces your score.
5. Press **Again!** to restart.

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla ES6)

# 🎯 Modal Popup Project

A simple **JavaScript project** that demonstrates how to create and manage a **modal popup window**.  
The modal can be opened with a button click and closed in multiple ways, making it a great beginner-friendly DOM manipulation exercise.

---

## 🚀 Features

- Open modal by clicking any **"Show Modal"** button.
- Close modal in **3 different ways**:
  - Click the **× (close button)**
  - Click outside on the **overlay**
  - Press the **Escape (Esc)** key
- Uses `classList.add()`, `classList.remove()`, and `classList.contains()` for DOM manipulation.

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript (ES6+)**

# 🎲 Pig Game

A fun two-player dice game built with **HTML, CSS, and JavaScript**.

## 🚀 How to Play

- Each player takes turns rolling a dice.
- The rolled number is added to the **current score**.
- If a player rolls **1**, they lose their current score and the turn switches.
- Players can **Hold** to add their current score to the total score and pass the turn.
- The first player to reach **20 points** wins the game! 🎉

## 🕹️ Features

- Two-player mode 👥
- Dynamic dice roll with images 🎲
- Active player highlight 🔥
- Win state with visual indicator 🏆
- Restart game button for quick replay
